# BugPro - Advanced Bug Tracker

BugPro is a modern, minimalistic bug tracking application designed for speed and simplicity. It features a futuristic glassmorphic UI, real-time issue management, and persistent local storage.

## Features

- **Futuristic UI**: Enhanced glassmorphism design with 3D interactive elements and smooth animations.
- **Theme Support**: Built-in Light and Dark modes.
- **Instant Updates**: Add, close, and delete issues without page reloads.
- **Responsive**: Fully optimized for desktop and mobile devices.
- **Secure**: Basic authentication flow to protect your dashboard.

## Quick Start

1. Open `index.html` in your web browser.

3. Start managing your tasks and bugs instantly!

## Technology Stack

- **HTML5**: Semantic and accessible structure.
- **Vanilla CSS**: Custom properties (variables) for theming and specialized layouts.
- **Vanilla JavaScript**: Lightweight logic with no heavy framework dependencies.
- **Local Storage**: Data persistence directly in your browser.

## Project Structure

- `index.html`: Main dashboard.
- `login.html`: Authentication page.
- `styles.css`: Core design system and utilities.
- `main.js`: Application logic.
